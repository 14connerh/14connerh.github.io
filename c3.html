Boolean Logic

Something that is used a lot in a program is Boolean Logic.

We often use this when typing things such as 'IF True:' etc.

In this chapter we looked at different methods of using Boolean Logic, including making flowcharts showing the diagrams, and making truth tables.









Notes

For truth tables, there are gates. These have an input and an output.



NOT gate - also known as inverter. The output is the opposite of the input, e.g 0 input = 1 output, or TRUE input = FALSE output.



AND gate - two inputs. If both are TRUE or both are 1, the output is TRUE or 1. Any other combination is FALSE, or 2.



OR gate - two inputs. If both inputs are FALSE, or 0, then the output is false. Any other combination has a TRUE, or 1 output.
