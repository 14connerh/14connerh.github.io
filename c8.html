Iterations

Another fundamental part of computers and the logic behind them is Iterations.

When you iterate something, it is repeated, usually until a condition is met.

In this chapter we looked at different methods of using iterations is python, such as WHILE, IF, or DO UNTIL.









Notes

A variable is something which has data assigned to it - either an integar or text. (E.G: age = 14) 'age' is the variable name, and 14 is the assigned data.



When planning a program, we can use flowcharts or pseudocode. There are examples of both on the document above. Pseudocode is structured English.



In flowcharts, you need to start and end using a rounded rectangle. For inputs, you need to use a parallelogram, and for an instruction, you can use a rectangle. For an 'if' or 'while' statement, a diamond shape needs to be used. Arrows are also used to show the 'flow' of the chart.



A constant is something which has data assigned to it, just like a variable, however the data cannot change; it is constant.



There are three types of operators, or signs that are used. Arithmetic operators include '+' or '-'. Relational operators include greater than and lesser than symbols. Finally, Boolean operators are 'and', 'not' and 'or'.



Last of all, we looked at the most effective ways to use IF statements. We should try use ELIF instead of IF, because ELIF is more efficient, due to ending iteration after finding the correct answer. Using many IF statements means that even if the answer has been found, if will still go through the other statements, which wastes time.
